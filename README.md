# To-do-list

To-do-list is a modern, dark-themed desktop to-do app built with Python and Tkinter.  
Itâ€™s designed for productivity, featuring tasks, categories, and visual checkboxes for done/undone tasks. The app is fully packaged and ready for distribution.

---

## Features

- **Add Tasks**: Quickly add new tasks with a title and optional category.  
- **Categories**: Organize tasks by color-coded categories.  
- **Done/Undone**: Check tasks off with visual animations.  
- **Filter**: View tasks by category or completed status.  
- **Dark UI**: Modern, clean dark theme with accent colors.  
- **Persistent Storage**: Tasks and categories are saved automatically in `data.json`.  
- **Vendable Ready**: Can be packaged into a single `.exe` using PyInstaller.  

---

## Requirements

- Python 3.9+  
- Tkinter (usually included with Python)  

Optional:
- PyInstaller (for packaging `.exe`)  

---

## Installation

1. Clone or download the repository.  
2. Ensure Python is installed.  
3. Install dependencies (if any):  
``` pip install pyinstaller ```
